有效沟通与反馈

by · 2020年09月26日 · 488 Words · ~1min reading time | Improve on

沟通流程模式

-> sender 
-> encoding 
-> method for sending message 
-> deconding
-> receiver
-> response
-> feedback

encoding & deconding process include noise

noise

  1. past experiences
  2. culture
  3. attitude
  4. values
  5. knowldege
  6. feelings

有效沟通技巧

->准确表达
->积极倾听
->有效反馈

准确表达—金字塔原理

信息是有组织、有结构的 先结论 后论据

积极倾听

听的层次

  1. 同理心地倾听
  2. 专注的听
  3. 选择性听
  4. 假装听
  5. 听而不闻

主动倾听四部曲

  1. 容许表达不同的意见，促进对方表达意愿
  2. 进一步获取更多信息
  3. 响应： 让对方知道自己在听，拉近彼此距离
  4. 重述： 确认自己没有逻辑错误

有效反馈

有效反馈三要素

  1. 要描述，不要判断
  2. 侧重现象，不攻击性格
  3. 要特指具体行为

反馈的两种类型

正面反馈

表扬团队时突出对个体的表扬
真诚的表扬（纯粹的正面反馈）
表扬行为价值
表扬要及时

STAR反馈法

S/T 情况saturation&任务task
A   行动action
R   结果result（确切说了什么，以及为何这样做会有效）

纠正性反馈

STARAR反馈法

S/T 情况saturation&任务task
A   行动action
R   结果result（确切说了什么，以及为何这样做会有效）
A   建议行动alternative action（本应该可以说或做什么）改进型反馈
R   预期结果 result 为何建议的行动或行为可能会更加有效
